### V16 核心流程
```
1. 解析MD → h0/h1/h2 + content/images/video/audio/link
2. 分析模板 → 三维索引 (h3_count, img_count, media_type)
3. 构建页面计划 → 模板匹配 + 轮换
4. 克隆幻灯片（XML深拷贝）
5. 填充内容
   ├─ 文本 → 拼音表格（混合字体）
   ├─ 图片 → 按位置替换（URL下载+缓存）
   ├─ 视频 → ffmpeg封面 + add_movie
   ├─ 音频 → add_movie
   └─ 链接 → click_action.hyperlink
6. 清除未使用占位符
7. 删除多余模板页
8. 保存
```

## 已知限制

| 问题 | 说明 |
|------|------|
| 模板组合覆盖不全 | 部分 (文本框数, 图片数, 媒体类型) 组合无对应模板，fallback到第一个正文模板 |
| 音频依赖系统能力 | 部分环境 add_movie 嵌入音频可能失败 |
| ffmpeg 为可选 | 无 ffmpeg 时视频封面提取失败，但不影响视频嵌入 |
