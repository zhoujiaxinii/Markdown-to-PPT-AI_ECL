## 一、md_parser.py — Markdown解析器

### 1.1 核心方法

#### parse() — 主解析方法

逐行扫描MD文件，按前缀识别内容类型：

| 前缀 | 识别为 | 存入字段 |
|------|--------|---------|
| `# ` (非`##`) | 一级标题 h0 | `result['h0']` |
| `## ` (非`###`) | 二级标题 h1 | `result['h1']` |
| `### ` (非`####`) | 三级标题 h2 | `result['h2']` |
| `#### ` | 四级标题 → 作为文本 | `current_content` |
| `![...](url)` | 图片（排除mp3/mp4） | `{type:'image'}` |
| `![...](...mp4)` | 视频 | `{type:'video'}` |
| `![...](...mp3)` | 音频 | `{type:'audio'}` |
| `http...mp4` | 纯视频链接 | `{type:'video'}` |
| `http...mp3` | 纯音频链接 | `{type:'audio'}` |
| `http...` | 链接 | `{type:'link'}` |
| 普通文本 | 文本 | `{type:'text'}` |

**换行处理**: 连续文本行用 `\n` 合并到同一个文本框（`||` 才是文本框分隔符）。

#### _parse_content() — 内容解析

对每个小节的 content_list 进行二次解析：

```
text 含 || → split('||') → 多个文本框
text 不含 || → 1个文本框

图片 → images: []
视频 → video: url | None
音频 → audio: url | None
链接 → link: url | None
```

#### get_word_limit_tip() — 字数/文本框提示

返回格式：`"1-50字（简短） | 1个文本框（合适）"`

- 字数分级：0 / 1-50 / 51-100 / 101-200 / 201-300 / 301-400 / 400+
- 文本框分级：0 / 1（合适）/ 2（合适）/ 3（合适）/ 4（已满）

#### _analyze_templates() — 三维模板索引

**章节页识别**: 包含 `{{h1_0}}` 但不含 `{{h2_0}}`
**正文页索引**: 包含 `{{h2_0}}` → 按 (h3数量, 图片数量, 媒体类型) 建立三维索引

```python
self.content_index = {
```

### 2.8 拼音表格

#### _parse_pinyin(text)

返回三元组: `(py_list, ch_list, english_segments)`

```
中文字符 → pypinyin取拼音 + 汉字
英文/数字/符号 → 空拼音 + 字符
连续英文段 → 记录 start/end/text 用于合并
换行符 → 保留为 '\n'
```

#### _create_pinyin_table() — 多行支持

```
单行文本 → _create_single_pinyin_table()
多行文本 → 按换行分割 → 每行独立创建表格 → 垂直排列
  每行高度 = 总高度 / 有效行数 × 0.9 (10%边距防止重叠)
```

#### _create_single_pinyin_table() — 核心表格创建

**中文内容**（2行×N列）:
```
表格结构：拼音行 + 汉字行
列宽 = 拼音长度 × fs × 0.5 × emu + fs × 0.6 × emu
英文段合并到一个单元格，宽度 = fs × 0.6 × len(text) + 边距
字号超出总宽度 → 按比例缩小（最小10pt）
```

**纯英文内容**（1行×1列）:
```
合并单元格，保留原始空格
字号 = 22pt（纯英文固定）
```

### 2.9 占位符查找

#### _find_all_placeholders(slide)

递归遍历所有 shape（包括 GROUP 内的子 shape），提取 `{{name}}` 格式的占位符。

```python
def _find_placeholders_recursive(self, shapes, ph, path=''):
    for s in shapes:
        if s.has_text_frame:
            for m in re.findall(r'\{\{(\w+)\}\}', s.text_frame.text):
                ph.setdefault(m, s)
        if s.shape_type == 6:  # GROUP
            self._find_placeholders_recursive(s.shapes, ph)
```

### 2.10 文本填充

#### _fill(slide, name, content, fs, bold)

核心逻辑分支：


### 2.11 主生成流程 — generate()

```
1. 构建页面计划
   ├─ 封面页（h0 → cover模板）
   ├─ 遍历每个 h2
   │   ├─ 章节页（h1 → section模板，仅章节切换时）
   │   └─ 正文页（content模板，按三维匹配）
   └─ 结束页（end模板）

2. 分配幻灯片（先克隆）
   ├─ 模板首次使用 → 直接引用原始 rId
   └─ 模板复用 → _clone_slide() 创建副本

3. 填充内容（克隆完成后）
   ├─ _fill_slide(slide, 'cover', title)
   ├─ _fill_slide(slide, 'section', section_name)
   ├─ _fill_slide(slide, 'content', h2_data)
   │   ├─ _fill h2_0 (小节标题, 28pt, bold)
   │   ├─ _fill h3_N (文本内容, 拼音表格)
   │   ├─ _embed_media (音视频)
   │   ├─ _embed_link (链接)
   │   └─ _replace_images (图片)
   └─ 结束页打印

4. 删除未使用的模板页
5. 按最终顺序重排 sldIdLst
6. prs.save(output)
```

## 三、关键算法

### 3.1 动态列宽计算

```python
# 字号转EMU (1pt = 12700 EMU, 1英寸 = 914400 EMU)
emu = lambda pt: int(pt * 914400 / 72)

# 中文字符列宽
py_len = len(pinyin_text)  # 拼音字符数
col_width = emu(fs * 0.5) * py_len + emu(fs * 0.6)

# 英文段列宽
en_width = emu(fs * 0.6) * len(en_text) + emu(fs * 0.3) * (len-1) + emu(fs * 0.3)
```

### 3.2 自动缩放

```python
total_width = sum(all_col_widths)
if total_width > available_width:
    scale = available_width / total_width
    new_fs = max(int(fs * scale), 10)  # 最小10pt
    # 重新计算所有列宽
```

### 3.3 表格位置对齐

```python
# 遵循占位符的原始对齐方式
if alignment == PP_ALIGN.LEFT:    table_left = left
if alignment == PP_ALIGN.CENTER:  table_left = left + (width - tw) // 2
if alignment == PP_ALIGN.RIGHT:   table_left = left + width - tw
```
