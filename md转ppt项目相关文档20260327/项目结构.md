## 核心模块说明

### 1. md_parser.py — Markdown解析器

**路径**: `src/utils/md_parser.py`
**行数**: 238行

**职责**: 将 Markdown 文件解析为结构化字典，供 PPTGenerator 使用。

**类定义**:
```python
class MDParser:
    def __init__(self, md_path)
    def parse(self) -> dict                    # 主解析方法
    def _parse_content(self, content_list)     # 内容列表解析
    def _create_h2_item(self, title, ...)      # 创建小节对象
    def get_word_limit_tip(self, ...)          # 字数/文本框提示
```

**输出结构**:
```python
{
    'h0': ['课程标题'],
    'h1': ['章节1', '章节2'],
    'h2': [{
        'title': '小节名',
        'section': '所属章节',
        'content': ['文本1', '文本2'],   # || 分隔的文本框
        'images': ['url1', 'url2'],
        'video': 'url' | None,
        'audio': 'url' | None,
        'link': 'url' | None,
        'char_count': 50,
        'word_limit_tip': '1-50字（简短） | 1个文本框（合适）'
    }]
}
```

---

### 2. ppt_generator.py — PPT生成器

**路径**: `src/utils/ppt_generator.py`
**行数**: 1568行

**职责**: 根据解析结果 + 模板PPT 生成输出PPT。

**核心类 PPTGenerator 的方法分组**:

| 分组 | 方法 | 说明 |
|------|------|------|
| 初始化 | `__init__`, `_analyze_templates` | 加载模板，建立三维索引 |
| 模板匹配 | `_match_content_template` | (文本框数, 图片数, 媒体类型) 匹配 |
| 幻灯片操作 | `_clone_slide`, `_get_slide_by_rId` | XML深拷贝 + rId追踪 |
| 图片替换 | `_replace_image`, `_replace_images_on_slide`, `_replace_image_from_url` | 在线下载 + 本地路径 |
| 音频嵌入 | `_embed_audio`, `_embed_audio_on_slide`, `_resolve_audio_path` | MP3嵌入 |
| 视频嵌入 | `_embed_media_to_placeholder`, `_embed_media_file`, `_embed_media_from_url` | MP4嵌入 + 封面提取 |
| 链接嵌入 | `_embed_link` | click_action.hyperlink |
| 拼音表格 | `_parse_pinyin`, `_create_pinyin_table`, `_create_single_pinyin_table` | 2×N透明表格 |
| 占位符 | `_find_all_placeholders`, `_find_placeholders_recursive` | 递归查找（含GROUP） |
| 文本填充 | `_fill`, `_set_mixed_font`, `_adjust_font_size_to_fit` | 混合字体 + 自动字号 |
| 布局 | `_center_text`, `_center_horizontal` | 居中对齐 |
| 生成 | `generate`, `_fill_slide` | 主流程 |

## 数据流

```
MD文件 ──→ MDParser.parse() ──→ 结构化字典
                                      │
模板PPT ──→ PPTGenerator.__init__()   │
         └── _analyze_templates()     │
              (三维索引)               │
                                      ▼
                              PPTGenerator.generate()
                                      │
                    ┌─────────────────┼─────────────────┐
                    ▼                 ▼                 ▼
              _clone_slide()    _fill_slide()    _replace_images()
              (XML深拷贝)      (拼音表格填充)    (图片URL替换)
                    │                 │                 │
                    └─────────────────┼─────────────────┘
                                      ▼
                               prs.save(output)
                                      │
                                      ▼
                              输出 .pptx 文件
```
