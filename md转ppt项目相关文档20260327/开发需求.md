# 开发需求

> 更新日期：2026-03-27
> 基于代码版本：V16

## 项目信息
- 项目名称：Markdown to PPT AI_ECL
- 项目目标：将 Markdown 文档自动转换为带拼音标注的 PPT 课件
- 使用场景：Echineselearning 对外汉语教学平台
- 当前版本：V16

---

## 整体逻辑

```
输入：1个 Markdown 文件 + 1个 PPT 模板
输出：1个填充好内容的 PPT 文件

┌─────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ MD 文件  │ →  │ MD解析器  │ →  │ PPT生成器 │ →  │ 输出PPT  │
└─────────┘    └──────────┘    └──────────┘    └──────────┘
                    ↑                ↑
                解析3级标题       读取PPT模板
                提取文本/图片     三维模板匹配
                音视频/链接       填充拼音表格
```

## R1: Markdown解析

| MD语法 | 含义 | PPT对应 |
|--------|------|---------|
| `# 标题` | 课程标题 | 封面页 (48pt) |
| `## 标题` | 章节名 | 章节页 (66pt) |
| `### 标题` | 小节名 | 正文页标题 (28pt bold) |
| `文本` | 正文 | 正文页文本框 |
| `A\|\|B` | 多文本框 | 同一页多个文本框（最多4个） |
| `![图](url)` | 图片 | 嵌入图片（最多5张/页） |
| `http...mp3` | 音频 | 嵌入音频 |
| `http...mp4` | 视频 | 嵌入视频 |
| `http...` | 链接 | 可点击链接 |

**换行**：MD中的换行 `\n` 合并为同一文本框内容，`||` 才是文本框分隔符。

---

## R2: PPT页面结构

```
封面页 (1页) — h0_0
├─ 章节A
│  ├─ 章节页 (1页) — h1_0
│  ├─ 正文页 — h2_0 + h3_* (+ media)
│  └─ 正文页 ...
├─ 章节B
│  ├─ 章节页 (1页)
│  └─ 正文页 ...
└─ 结束页 (1页) — "谢谢"
```

**实现**：XML深拷贝幻灯片，先克隆后填充，rId追踪排序，未使用模板页删除。

---

## R3: 拼音标注

- 2行×N列透明表格（无边框、无底色）
- 第1行：拼音（SimHei）
- 第2行：汉字（SimHei）
- 列宽动态计算（基于拼音长度和字号）
- 自动缩放（最小10pt）
- 纯英文：1行×1列合并单元格（22pt）
- 多行文本：每行独立创建表格，垂直排列

---

## R4: 三维模板匹配

```python
# 索引结构
content_index[(h3_count, img_count, media_type)] = [页索引列表]

# 匹配优先级
1. (text_count, image_count, media_type)  精确匹配
2. (text_count, image_count, None)        忽略媒体类型
3. 无匹配 → fallback第一个正文模板 + 红色提示
```

**媒体类型**：video > audio > link > None

---

## R5-R6: 背景保留与图片嵌入

- XML深拷贝保留模板所有元素
- 图片按位置排序（top→left），顺序替换
- 在线URL：MD5缓存到 `.tmp_images/`
- 路径解析：绝对路径 → MD目录 → 模板目录 → 工作目录

---

## R7-R8: 音视频嵌入

| 类型 | 占位符 | 嵌入方式 | 封面 |
|------|--------|---------|------|
| 音频 | `{{audio}}` | add_movie | 无 |
| 视频 | `{{video}}` | add_movie | ffmpeg提取第一帧 |

**ffmpeg命令**：`ffmpeg -i video.mp4 -ss 00:00:00.5 -vframes 1 -q:v 2 -y poster.jpg`

**失败处理**：显示红色错误文本

---

## R13: 链接嵌入

- 占位符：`{{link}}`
- 显示："🔗 点击打开游戏链接进行游玩"
- 字体：18pt SimHei 蓝色
- 超链接：`shape.click_action.hyperlink.address = url`

---

## R15: 字号规则

| 文本框数 | 条件 | 字号 |
|----------|------|------|
| 1 | 无图片，中文≤10字 | 40pt |
| 1 | 无图片，中文>10字 | 28pt |
| 2 | 无图片，中文>10字 | 28pt |
| 2 | 其他 | 36pt |
| 3 | - | 30pt |
| 4 | - | 24pt |

自动缩放：超出文本框时缩小（最小8pt）。

---

## 内容限制

| 元素 | 限制 |
|------|------|
| 文本框/页 | 最多4个 (`\|\|`分隔) |
| 图片/页 | 最多5张 |
| 标题层级 | 3级 (`#`→`##`→`###`) |
| 拼音最小字号 | 10pt |
