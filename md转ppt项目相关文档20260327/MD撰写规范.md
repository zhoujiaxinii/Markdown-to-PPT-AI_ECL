# MD撰写规范

> 更新日期：2026-03-27
> 基于代码版本：V16

---

## 一、文档结构（3级标题）

| 级别 | 语法 | PPT对应 | 说明 |
|------|------|---------|------|
| 一级 | `# 标题` | 封面页 | 课程标题，48pt，居中 |
| 二级 | `## 标题` | 章节页 | 章节名，66pt，居中 |
| 三级 | `### 标题` | 正文页 | 小节名，28pt，加粗 |

**每个 `###` 小节生成一页PPT。**

---

## 二、文本框

### 单文本框
```markdown
### 学一学
这是文本内容。
```

### 多文本框（最多4个）

用 `||` 分隔：
```markdown
### 学一学
A：这是什么颜色？||B：这是白色。
```

### 换行

MD 中的换行符 `\n` 保留在同一文本框内（只有 `||` 才分隔文本框）：
```markdown
### 学一学
第一行内容
第二行内容
```
→ 生成一个包含两行文本的文本框。

### 字数提示

| 字数范围 | 推荐文本框数 |
|----------|-------------|
| 0-50字 | 1个 |
| 51-100字 | 1-2个 |
| 101-200字 | 2-3个 |
| 201-300字 | 3-4个 |
| 301-400字 | 3-4个 |
| 400字+ | 建议分页 |

---

## 三、图片（0-5张/页）

```markdown
![图片描述](https://example.com/image.jpg)
```

- 支持在线 URL（自动下载缓存）
- 支持本地相对路径（相对于 MD 文件目录）
- 模板中的图片按位置顺序替换

---

## 四、音频（MP3）

```markdown
### 听录音
请听录音判断对错。
https://example.com/audio.mp3
```

或与文本同页：
```markdown
### 学一学
A：听录音回答问题。||B：我听懂了。
https://example.com/audio.mp3
```

**规则**：
- URL 必须以 `.mp3`、`.wav`、`.m4a` 结尾
- 匹配含 `{{audio}}` 占位符的模板页
- 嵌入 PPT，显示图标，点击可播放

---

## 五、视频（MP4）

```markdown
### 看视频
看视频回答问题。
https://example.com/video.mp4
```

或与文本同页：
```markdown
### 学一学
看视频回答问题。||A：是的。
https://example.com/video.mp4
```

**规则**：
- URL 必须以 `.mp4`、`.avi`、`.webm` 结尾
- 匹配含 `{{video}}` 占位符的模板页
- 嵌入 PPT，自动提取第一帧作为封面

---

## 六、网页链接

```markdown
### 点一点
点击链接玩游戏。
https://example.com/game
```

**规则**：
- 任意非音视频 URL 均识别为链接
- 匹配含 `{{link}}` 占位符的模板页
- 点击后用默认浏览器打开

---

## 七、字体规则

### 字体
- 中文：SimHei（黑体）
- 英文/数字：Arial
- 混合内容：按段自动分割，分别设置字体

### 字号（自动计算）

| 文本框数 | 条件 | 字号 |
|----------|------|------|
| 1个 | 无图片，中文≤10字 | 40pt |
| 1个 | 无图片，中文>10字 | 28pt |
| 2个 | 无图片，中文>10字 | 28pt |
| 2个 | 其他 | 36pt |
| 3个 | - | 30pt |
| 4个 | - | 24pt |

文本超出文本框时自动缩小（最小8pt）。

---

## 八、拼音标注

- 所有汉字自动添加拼音，显示在汉字正上方
- 使用透明表格实现精确对齐
- 纯英文行合并为单个单元格（22pt）
- 支持多行文本（每行独立创建拼音表格）

---

## 十、注意事项

| 项目 | 限制 |
|------|------|
| 标题层级 | 一级 `#` → 二级 `##` → 三级 `###` |
| 文本框/页 | 最多4个（`||` 分隔） |
| 图片/页 | 最多5张 |
| 音频 | 以 `.mp3`/`.wav`/`.m4a` 结尾的 URL |
| 视频 | 以 `.mp4`/`.avi`/`.webm` 结尾的 URL |
| 链接 | 非音视频 URL |
| 拼音 | 自动生成，无需手动 |
