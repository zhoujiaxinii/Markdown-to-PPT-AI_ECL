# 开发需求

## 项目概述
- 项目名称：Markdown to PPT AI_ECL
- 项目目标：使用AI智能将Markdown文本转换为PPT文件
- 开发平台：Echineselearning平台

## 具体需求

### 1. 拼音格式优化
- **需求描述**：汉字转换为拼音时，拼音应显示在汉字上方，并且与汉字一一对齐
- **当前状态**：当前是 汉字（拼音） 格式
- **期望效果**：类似 PPT 拼音指南插件的效果

### 2. 音视频嵌入支持
- **需求描述**：当前版本仅支持在 PPT 中放置链接，需要实现音视频文件直接嵌入到 PPT 中播放
- **支持格式**：mp3、mp4、wav 等主流音视频格式

### 3. 动态模板匹配逻辑
- **需求描述**：根据 Markdown 中每页的实际内容数量，自动匹配对应的模板页面
- **匹配规则**：
  - 1个文本 → 匹配"单文本"模板
  - 1文本 + 1图片 → 匹配"文图"模板
  - 2张图片 → 匹配"双图"模板
  - 文本 + 图片 + 音频 → 匹配"多媒体"模板

### 4. 项目运行框架图
- **需求描述**：在md文件中加入项目运行框架图
- **实现方式**：使用mermaid语法绘制流程图

## 技术要求
- 使用Python语言开发
- 使用Flask框架构建Web服务
- 使用python-pptx库生成PPT
- 使用pypinyin库处理拼音
- 支持Markdown解析
- 支持音视频文件嵌入
- 支持动态模板匹配
