## 核心文件说明

### 1. md_parser.py - Markdown解析器

**功能**：解析MD文件，提取结构化内容

**主要方法**：
```python
class MDParser:
    def parse(self) -> dict:
        """解析MD文件，返回结构化数据"""
        
    def _parse_content(self, content_list) -> dict:
        """解析内容列表，提取文本、图片、视频、音频"""
        
    def get_word_limit_tip(self, char_count, text_box_count) -> str:
        """返回字数限制提示"""
```

**输出结构**：
```python
{
    'h0': ['课程标题'],
    'h1': ['章节1', '章节2'],
    'h2': [
        {
            'title': '小节名',
            'content': ['文本1', '文本2'],
            'images': ['url1', 'url2'],
            'video': 'url',
            'audio': 'url',
            'char_count': 50,
            'word_limit_tip': '1-50字（简短）| 1个文本框（合适）'
        }
    ]
}
```

### 2. ppt_generator.py - PPT生成器

**功能**：根据解析结果生成PPT，包含拼音表格逻辑

**主要方法**：
```python
class PPTGenerator:
    def generate(self) -> str:
        """生成PPT，返回输出路径"""
        
    def _create_pinyin_table(self, slide, left, top, width, height, text, font_size):
        """创建拼音-汉字对齐表格"""
        
    def _hide_table_style(self, table):
        """隐藏表格边框和底色"""
        
    def _parse_pinyin_chars(self, text):
        """解析文本，返回拼音和字符列表"""
        
    def _replace_placeholder_with_pinyin_table(self, slide, placeholder_type, content, font_size):
        """替换占位符为拼音表格"""
```

**拼音表格实现**：
- 2行 x N列结构
- 动态列宽计算
- 自动字号缩放
- 禁止换行
- 无边框、无底色

```
MD文件 → MDParser.parse() → 结构化数据 → PPTGenerator.generate() → PPT文件
```
