# PPT 模板编写规则说明

> 更新日期：2026-03-06
> 基于 3.6 模板.pptx (32页)

## 一、占位符命名规范

### 1. 文本占位符

| 占位符 | 用途 | 示例 |
|--------|------|------|
| `{{h0_0}}` | 封面标题 | 课程名称 |
| `{{h1_0}}` | 章节标题 | 章节名称 |
| `{{h2_0}}` | 小节标题 | 小节名称 |
| `{{h3_0}}` | 正文内容框1 | 文本内容 |
| `{{h3_1}}` | 正文内容框2 | 文本内容（可选） |
| `{{h3_2}}` | 正文内容框3 | 文本内容（可选） |
| `{{h3_3}}` | 正文内容框4 | 文本内容（可选） |

### 2. 媒体占位符

| 占位符 | 用途 | 说明 |
|--------|------|------|
| `{{audio}}` | 音频占位符 | 嵌入音频文件位置 |
| `{{video}}` | 视频占位符 | 嵌入视频文件位置 |

### 3. 图片占位符

模板中的图片直接作为占位符使用，无需命名。

## 二、页面类型定义

### 1. 封面页
- **占位符**：`{{h0_0}}`（课程标题）
- **可选**：背景图片（1张）

### 2. 章节页
- **占位符**：`{{h1_0}}`（章节标题）
- **可选**：章节装饰图

### 4. 正文页
- **必须**：`{{h2_0}}`（小节标题）
- **内容框**：`{{h3_0}}` ~ `{{h3_3}}`（最多4个正文框）
- **可选**：`{{audio}}`（音频图标位置）
- **可选**：`{{video}}`（视频图标位置）
- **可选**：正文图片（0-5张）

### 5. 结束页
- **无需占位符**
- **通常**：包含"谢谢"或"xiè xie"文字（2张图片）

## 四、音视频占位符使用

### 方案 A：独立占位符（推荐）

在正文中添加 `{{audio}}` 或 `{{video}}` 占位符：

```
+----------------------------------+
| {{h2_0}}                    标题 |
+----------------------------------+
| {{h3_0}}              | {{audio}} |
| 正文内容...          | [🔊]     |
+----------------------------------+
| {{h3_1}}                        |
| 第二段正文...                    |
+----------------------------------+
```

生成器会自动：
1. 识别 `{{audio}}` 并在对应位置嵌入音频图标
2. 识别 `{{video}}` 并在对应位置嵌入视频播放器
3. 清除占位符文本

### 方案 B：文本标记（备选）

当模板不支持音视频占位符时，生成器会自动创建额外文本框显示标记：
- 音频：`[🔊 点击播放: xxx.mp3]`
- 视频：`[🎬 点击播放: xxx.mp4]`

## 五、图片处理规则

### 自动替换
- 模板中的图片作为占位符
- MD 中的图片按顺序替换（按位置从上到下、从左到右）
- 保持原图尺寸和位置

### 模板匹配逻辑
```
输入：文本框数量 + 图片数量 + 音频/视频
匹配：精确匹配 → 同框多图 → 跨框匹配 → 任意
```

2. `{{h3_0}}` ~ `{{h3_3}}` - 正文内容
3. `{{audio}}` - 音频
4. `{{video}}` - 视频

## 八、创建新模板步骤

1. **确定页面类型**：封面/章节/正文/结束
2. **添加文本占位符**：按上述规范命名
3. **放置图片占位符**：按需放置
4. **（可选）添加音视频占位符**：`{{audio}}`、`{{video}}`
5. **设置样式**：字体、颜色、背景等
6. **测试**：使用 test_sample.md 验证生成效果

## 十一、常见问题

| 问题 | 解决 |
|------|------|
| 音视频标记不显示 | 使用方案A添加 `{{audio}}`/`{{video}}` 占位符 |
| 文本框数量不够 | 选用 2+ 文本框的模板 |
| 图片位置不对 | 检查模板图片顺序，确保与 MD 中图片顺序一致 |
| 拼音显示异常 | 确保使用透明表格方案，无边框底色 |
| 模板匹配失败 | 检查文本框数量+图片数量+音视频是否与模板一致 |
