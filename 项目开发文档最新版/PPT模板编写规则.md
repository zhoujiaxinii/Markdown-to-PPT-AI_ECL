# PPT 模板编写规则说明

> 更新日期：2026-03-11
> 基于 3.10 模板.pptx (37页)

## 一、占位符命名规范

### 1. 文本占位符

| 占位符 | 用途 | 示例 |
|--------|------|------|
| `{{h0_0}}` | 封面标题 | 课程名称 |
| `{{h1_0}}` | 章节标题 | 章节名称 |
| `{{h2_0}}` | 小节标题 | 小节名称 |
| `{{h3_0}}` | 正文内容框1 | 文本内容 |
| `{{h3_1}}` | 正文内容框2 | 文本内容（可选） |
| `{{h3_2}}` | 正文内容框3 | 文本内容（可选） |
| `{{h3_3}}` | 正文内容框4 | 文本内容（可选） |

### 2. 媒体占位符

| 占位符 | 用途 | 说明 |
|--------|------|------|
| `{{audio}}` | 音频占位符 | 嵌入音频文件，点击可播放 |
| `{{video}}` | 视频占位符 | 嵌入视频文件，提取第一帧作为封面 |
| `{{link}}` | 链接占位符 | 点击后用浏览器打开链接 |

### 3. 图片占位符

模板中的图片直接作为占位符使用，无需命名。

## 二、页面类型定义

### 1. 封面页
- **占位符**：`{{h0_0}}`（课程标题）
- **可选**：背景图片（1张）

### 2. 章节页
- **占位符**：`{{h1_0}}`（章节标题）
- **可选**：章节装饰图

### 3. 正文页
- **必须**：`{{h2_0}}`（小节标题）
- **内容框**：`{{h3_0}}` ~ `{{h3_3}}`（最多4个正文框）
- **可选**：`{{audio}}`（音频图标位置）
- **可选**：`{{video}}`（视频图标位置）
- **可选**：`{{link}}`（链接图标位置）
- **可选**：正文图片（0-5张）

### 4. 结束页
- **无需占位符**
- **通常**：包含"谢谢"或"xiè xie"文字

### 二维匹配规则
- 按 (文本框数, 图片数, 媒体类型) 三维索引匹配
- 媒体类型优先级：video > audio > link > None
- 如果没有匹配的媒体类型模板，使用无媒体模板

## 四、媒体占位符使用

### 4.1 占位符说明

- `{{audio}}` - 音频占位符，匹配后显示音频图标
- `{{video}}` - 视频占位符，匹配后显示视频（带封面）
- `{{link}}` - 链接占位符，点击后用浏览器打开

### 4.2 使用示例

在模板中添加占位符：

```
+----------------------------------+
| {{h2_0}}                    标题 |
+----------------------------------+
| {{audio}}                      |
| [🔊]                         |
+----------------------------------+
```

## 五、图片处理规则

### 自动替换
- 模板中的图片作为占位符
- MD 中的图片按顺序替换（按位置从上到下、从左到右）
- 保持原图尺寸和位置

## 七、创建新模板步骤

1. **确定页面类型**：封面/章节/正文/结束
2. **添加文本占位符**：按上述规范命名
3. **放置图片占位符**：按需放置
4. **（可选）添加媒体占位符**：`{{audio}}`、`{{video}}`、`{{link}}`
5. **设置样式**：字体、颜色、背景等
6. **测试**：使用测试文件验证生成效果

## 九、常见问题

| 问题 | 解决 |
|------|------|
| 音视频不显示 | 使用 `{{audio}}`/`{{video}}` 占位符 |
| 链接无法点击 | 使用 `{{link}}` 占位符 |
| 文本框数量不够 | 选用 2+ 文本框的模板 |
| 图片位置不对 | 检查模板图片顺序 |
| 拼音显示异常 | 确保使用透明表格方案 |
| 模板匹配失败 | 检查文本框数量+图片数量+媒体类型是否与模板一致 |
| 英文显示异常 | 纯英文行会自动合并单元格 |
