# 开发需求

> 更新日期：2026-03-11

## 项目信息
- 项目名称：Markdown to PPT AI_ECL
- 项目目标：将 Markdown 文档自动转换为带拼音标注的 PPT 课件
- 使用场景：Echineselearning 对外汉语教学平台
- 当前版本：V17

---

## 整体逻辑

```
输入：1个 Markdown 文件 + 1个 PPT 模板
输出：1个填充好内容的 PPT 文件

流程：
┌─────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ MD 文件  │ →  │ MD解析器  │ →  │ PPT生成器 │ →  │ 输出PPT  │
└─────────┘    └──────────┘    └──────────┘    └──────────┘
                    ↑                ↑
                解析3级标题       读取PPT模板
                提取文本/图片     匹配模板页
                音视频/链接       填充拼音表格
```

## R1：Markdown 解析

### 需求描述
将 Markdown 文件解析为结构化数据，供 PPT 生成器使用。

### 规则

| MD语法 | 含义 | PPT对应 |
|--------|------|---------|
| `# 标题` | 课程标题 | 封面页标题 |
| `## 标题` | 章节名 | 章节页 |
| `### 标题` | 小节名 | 正文页标题 |
| `文本内容` | 正文 | 正文页文本框 |
| `文本A\|\|文本B` | 多文本框 | 同一页内多个文本框（最多4个） |
| `![图片](url)` | 图片 | 嵌入图片（最多5张/页） |
| `https://xxx.mp3` | 音频文件 | 嵌入音频 |
| `https://xxx.mp4` | 视频文件 | 嵌入视频（带第一帧封面） |
| `https://xxx` | 链接 | 点击后用浏览器打开 |

---

## R2：PPT 页面结构

### 需求描述
生成的 PPT 必须按固定结构排列页面。

### 页面顺序规则

```
封面页（1页）
  └─ 显示课程标题（# 标题）

章节 A 的内容
  ├─ 章节页（1页）── 显示章节名
  ├─ 正文页 1 ── 第一个 ### 小节
  ├─ 正文页 2 ── 第二个 ### 小节
  └─ ...

章节 B 的内容
  ├─ 章节页（1页）
  ├─ 正文页 ...
  └─ ...

结束页（1页）
  └─ 自动添加，显示"谢谢"（无需MD编写）
```

---

## R3：拼音标注与对齐

### 需求描述
所有汉字自动生成拼音，拼音显示在汉字**正上方**，一一精确对齐。

### 实现方案
使用 **2行 × N列透明表格**：
- 第1行：拼音（Arial字体）
- 第2行：汉字（SimSun字体）
- 表格无边框、无背景色
- 列宽根据拼音长度动态计算
- 字号超出宽度时自动缩小（最小10pt）
- 单元格禁止换行

### 英文处理
- 英文文本保留单词间的空格
- 纯英文行使用单行单列表格（合并单元格）

---

## R4：智能模板匹配

### 需求描述
根据每页 MD 内容的**文本框数量、图片数量、媒体类型**，自动从模板中选择最合适的页面。

### 匹配规则

| 页面类型 | 识别方式 | 占位符 |
|---------|---------|--------|
| 封面页 | 含 `{{h0_0}}` | h0_0 |
| 章节页 | 含 `{{h1_0}}`，不含 `{{h2_0}}` | h1_0 |
| 正文页 | 含 `{{h2_0}}` | h2_0 + h3_0~h3_3 |
| 结束页 | 含"谢谢"或"xiè xie" | 无 |

**正文页匹配逻辑**：
- 按 `{{h3_*}}` 占位符数量分类（1框、2框、3框、4框）
- 按图片数量精确匹配
- 按媒体类型匹配（video > audio > link > None）
- MD 内容有 N 个文本框 → 选 ≥N 的最小模板
- 多余的占位符自动清除

---

## R5：模板背景保留

### 需求描述
生成的 PPT 必须保留模板的**背景、图片、装饰元素**。

### 实现方案
- 在模板 PPT 上直接操作（不创建新PPT）
- 需要复用的模板页通过 **XML 深拷贝** 克隆
- 克隆保留：背景（来自slide layout）、图片、自动形状等
- 先克隆后填充，避免克隆到已填充的内容
- 用 rId 追踪每个幻灯片，确保删除和排序不出错

---

## R6-R8：媒体嵌入

### 音频（MP3）
- 可以**单独占一页**，也可以与文本框**同页**
- 匹配含 `{{audio}}` 占位符的模板页
- 嵌入音频，显示图标，点击可播放

### 视频（MP4）
- 可以**单独占一页**，也可以与文本框**同页**
- 匹配含 `{{video}}` 占位符的模板页
- 嵌入视频，提取第一帧作为封面

### 链接
- 匹配含 `{{link}}` 占位符的模板页
- 点击后用默认浏览器打开

## R9：字号规则

| 文本框数 | 条件 | 字体大小 |
|----------|------|----------|
| 1个 | 无图片，中文≤10字 | 40pt |
| 1个 | 无图片，中文>10字 | 28pt |
| 2个 | 无图片，中文>10字 | 28pt |
| 2个 | 其他 | 36pt |
| 3个 | - | 30pt |
| 4个 | - | 24pt |

## 内容限制

| 元素 | 限制 | 说明 |
|------|------|------|
| 文本框/页 | 最多4个 | 用 `\|\|` 分隔 |
| 图片/页 | 最多5张 | `![图片](url)` |
| 标题层级 | 3级 | `#` → `##` → `###` |
| 拼音最小字号 | 10pt | 超宽时自动缩小 |
