# 开发需求

> 更新日期：2026-03-04

## 项目信息
- 项目名称：Markdown to PPT AI_ECL
- 项目目标：使用AI智能将Markdown文本转换为PPT文件
- 开发平台：Echineselearning平台

---

## 功能需求

### 1. 拼音格式优化 ✅ 已完成

**需求描述**：汉字转换为拼音时，拼音应显示在汉字上方，并与汉字一一精确对齐

**实现方案**：
- 使用 **2行 x N列无边框、无底色表格** 实现精确对齐
- 每个拼音-汉字对在独立单元格中
- 列宽根据拼音长度动态计算
- 拼音间距：每个拼音后 + 1个字母宽度
- 自动缩放字号适应宽度，最小10号
- 禁止单元格换行

**效果示意**：
```
 zhè  shì  bái  sè
 这    是    白    色
```

---

### 2. 音视频嵌入支持 ✅ 已完成

**需求描述**：实现音视频文件直接嵌入到PPT中播放

**支持格式**：
- 音频：mp3、wav
- 视频：mp4

**实现方式**：
- Markdown中使用 `![音频](url)` 和 `![视频](url)` 标记
- 音频与文本同页，匹配带喇叭图标模板
- 视频单独一页，嵌入显示

---

### 3. 动态模板匹配 ✅ 已完成

**需求描述**：根据Markdown中每页的实际内容数量，自动匹配对应的模板页面

**匹配规则**：

| 内容类型 | 模板类型 |
|---------|---------|
| 1个文本框 | 单文本模板 |
| 1文本 + 1图片 | 文图模板 |
| 多张图片 | 多图模板 |
| 文本 + 音频 | 音频模板（带喇叭图标） |
| 视频 | 视频模板（单独一页） |

---

### 4. 内容限制

| 元素 | 限制 |
|------|------|
| 文本框/页 | 最多4个（用 `\|\|` 分隔） |
| 图片/页 | 最多5张 |
| 标题层级 | 3级（# → ## → ###） |

---

## 技术要求

- 使用Python语言开发
- 使用python-pptx库生成PPT
- 使用pypinyin库处理拼音
- 使用lxml处理XML样式
- 支持Markdown解析

---

## 验收标准

| 功能 | 验收标准 |
|------|---------|
| 拼音对齐 | 每个拼音精确对齐在对应汉字正上方 |
| 无边框底色 | 表格完全透明，无边框显示 |
| 字符间距 | 紧凑自然，拼音间距1个字母宽度 |
| 自动缩放 | 文本过长时自动缩小字号，最小10号 |
| 不换行 | 拼音和汉字都不出现折行 |
