## PPT生成流程（V10）

```
MD文档 → 解析 → 结构化数据 → 匹配模板 → 创建新PPT → 复制幻灯片 → 填充内容 → 输出PPT

核心改进：
- 创建新的PPT文件，而不是修改模板
- 按顺序复制模板幻灯片到新PPT
- 避免删除页面后索引混乱的问题

结构：
1. 封面页 (h0)
2. 目录页 (h1列表)
3. 遍历章节
   ├─ 章节页 (h1标题)
   └─ 正文页 (h2内容)
4. 结束页 (自动添加)
```
