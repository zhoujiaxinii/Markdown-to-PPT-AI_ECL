### V12 核心逻辑
```
1. 分析模板 → 识别封面/目录/章节/正文/结束页
2. 构建页面计划 → [(类型, 模板索引, 数据), ...]
3. 分配幻灯片（先克隆，不填充）
   ├─ 首次使用模板 → 直接用原始页
   └─ 重复使用模板 → XML深拷贝到末尾
4. 统一填充内容（所有克隆完成后）
5. 删除未使用的模板页（按rId精确定位）
6. 按最终顺序重排sldIdLst（按rId映射）
7. 保存输出
```
