### V17 核心逻辑
```
1. 解析MD → 提取h0/h1/h2，文本/图片/音视频/链接
   └─ 文本框：||分隔多框，换行合并为一框
2. 分析模板 → 识别封面/章节/正文/结束页
   └─ 正文模板按 (文本框数, 图片数, 媒体类型) 三维索引
3. 构建页面计划 → 匹配模板
4. 填充内容
   ├─ 文本 → 拼音表格替换占位符，支持换行
   │   └─ 英文文本保留空格，纯英文行合并单元格
   ├─ 图片 → 保持模板位置替换
   ├─ 视频 → 嵌入并提取第一帧作为封面
   ├─ 音频 → 嵌入PPT，显示图标，点击可播放
   └─ 链接 → 点击后用浏览器打开
5. 清除未使用的占位符（含{{...}}的文本框）
6. 字号计算：
   └─ 1-2框+无图片+中文>10 → 28pt
   └─ 其他按文本框数计算
7. 保存输出
```
