### V16 核心逻辑
```
1. 解析MD → 提取h0/h1/h2，文本/图片/音视频
   └─ 文本框：||分隔多框，换行合并为一框
2. 分析模板 → 识别封面/章节/正文/结束页
   └─ 正文模板按 (文本框数, 图片数) 二维索引
3. 构建页面计划 → 匹配模板
4. 填充内容
   ├─ 文本 → 拼音表格替换占位符，支持换行
   ├─ 图片 → 保持模板位置替换
   ├─ 视频 → 嵌入并提取第一帧作为封面
   └─ 音频 → 嵌入PPT
5. 清除未使用的占位符（含{{...}}的文本框）
6. 标题居中：一级水平居中，二级完全居中
7. 保存输出
```
