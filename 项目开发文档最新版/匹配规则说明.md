# 匹配规则说明

> 更新日期：2026-03-04
> 详细说明MD内容如何匹配到PPT模板的规则

---

## 一、页面类型识别

系统根据MD内容自动识别页面类型：

```
视频页面 > 音频页面 > 普通正文页面 > 章节页面 > 目录页面 > 封面
```

---

## 二、页面类型与模板匹配

### 2.1 封面页
- **识别条件**：`# 一级标题`
- **模板占位符**：`{{h0_0}}`
- **示例**：`# 说白色`

### 2.2 目录页
- **识别条件**：`## 二级标题` 列表
- **模板占位符**：`{{h1_0}}`, `{{h1_1}}`, ...

### 2.3 章节页
- **识别条件**：单独的 `## 二级标题`
- **模板占位符**：`{{h1_0}}`

### 2.4 普通正文页
- **识别条件**：`### 小节名` + 文本内容
- **模板占位符**：
  - 小节名：`{{h2_0}}`
  - 文本框：`{{h3_0}}`, `{{h3_1}}`, `{{h3_2}}`, `{{h3_3}}`
  - 图片：`{{img_0}}` ~ `{{img_4}}`

### 2.5 音频页面
- **识别条件**：`### 小节名` + `![音频](url)`
- **模板特征**：包含喇叭图标占位符 `{{audio_0}}`

### 2.6 视频页面
- **识别条件**：`### 小节名` + `![视频](url)`
- **模板特征**：包含视频占位符 `{{video_0}}`

---

## 三、文本框匹配规则

### 3.1 分隔符
使用 `||` 分隔多个文本框：

```markdown
### 学一学
第一段||第二段||第三段
```

### 3.2 占位符对应

| 文本框序号 | 占位符 |
|-----------|--------|
| 第1个 | `{{h3_0}}` |
| 第2个 | `{{h3_1}}` |
| 第3个 | `{{h3_2}}` |
| 第4个 | `{{h3_3}}` |

---

## 四、图片匹配规则

| 图片序号 | 占位符 |
|---------|--------|
| 第1张 | `{{img_0}}` |
| 第2张 | `{{img_1}}` |
| 第3张 | `{{img_2}}` |
| 第4张 | `{{img_3}}` |
| 第5张 | `{{img_4}}` |

---

## 五、音视频识别规则

### 5.1 音频识别
- **关键词**：`![音频](url)`
- **URL后缀**：`.mp3`
- **页面行为**：与文本同页

### 5.2 视频识别
- **关键词**：`![视频](url)`
- **URL后缀**：`.mp4`
- **页面行为**：单独一页

---

## 六、拼音生成规则

### 6.1 垂直对齐原则
- **核心要求**：每个拼音精确对齐到对应汉字的**正上方**
- 拼音和汉字**一一对应**

### 6.2 表格实现方案
使用 **2行 x N列** 的无边框、无底色表格：

```
原文：这是白色
实现：
 zhè  shì  bái  sè
 这    是    白    色
```

### 6.3 字符间距规则
- **拼音间距**：每个拼音后 + 1个字母宽度
- **汉字间距**：跟随对应拼音
- **列宽计算**：动态计算，不均分

### 6.4 字体设置

| 行 | 字体 | 说明 |
|----|------|------|
| 拼音行 | Arial | 英文字体 |
| 汉字行 | SimSun | 宋体 |
| 字号 | 相同 | 自动缩放 |

### 6.5 表格样式
- **边框**：完全隐藏
- **底色**：透明
- **行高**：拼音45%，汉字55%
- **换行**：禁止（`word_wrap = False`）

### 6.6 自动缩放
- 文本过长时自动缩小字号
- 最小字号：10号

---

## 七、模板匹配流程

```
1. 读取MD文档
   ↓
2. 解析标题层级
   ↓
3. 遍历每个小节(###)
   ├─ 有视频？→ 视频模板
   ├─ 有音频？→ 音频模板
   └─ 普通内容 → 正文模板
   ↓
4. 替换占位符
   ├─ 文本 → 拼音表格
   └─ 图片/音视频 → 对应元素
   ↓
5. 生成PPT
```

## 八、占位符完整列表

| 类型 | 占位符 | 说明 |
|------|--------|------|
| 封面 | `{{h0_0}}` | 课程标题 |
| 目录 | `{{h1_0}}`~`{{h1_9}}` | 章节列表 |
| 章节 | `{{h1_0}}` | 章节标题 |
| 小节 | `{{h2_0}}` | 小节标题 |
| 文本 | `{{h3_0}}`~`{{h3_3}}` | 文本框 |
| 图片 | `{{img_0}}`~`{{img_4}}` | 图片 |
| 音频 | `{{audio_0}}` | 音频文件 |
| 视频 | `{{video_0}}` | 视频文件 |
