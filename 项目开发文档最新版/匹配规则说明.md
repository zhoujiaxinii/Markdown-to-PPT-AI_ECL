# 匹配规则说明

> 更新日期：2026-03-04
> 详细说明MD内容如何匹配到PPT模板的规则

---

## 一、页面类型识别

系统根据MD内容自动识别页面类型：

```
视频页面 > 音频页面 > 普通正文页面 > 章节页面 > 目录页面 > 封面
```

---

## 二、智能模板匹配

### 2.1 匹配原则

**核心要求**：根据MD内容精确匹配模板，清除多余占位符

| MD内容 | 匹配模板 | 说明 |
|--------|---------|------|
| 1个文本框 | 文本框数=1的模板 | 清除多余文本框占位符 |
| 1文本框+1图片 | 文本框=1且图片≥1的模板 | 清除多余图片 |
| 2个文本框 | 文本框数=2的模板 | 清除多余文本框占位符 |
| N个文本框 | 文本框数≥N的最小模板 | 清除未使用的占位符 |

### 2.2 模板分类

模板分析时会自动识别：
- 封面页：包含 `{{h0_0}}`
- 章节页：包含 `{{h1_0}}` 且不包含 `{{h2_0}}`
- 正文页：包含 `{{h2_0}}`（统计文本框数和图片数）

### 2.3 匹配算法

```
对于每个MD小节：
1. 统计需要的文本框数 text_count
2. 统计需要的图片数 img_count
3. 在模板库中查找：
   - 模板文本框数 >= text_count
   - 模板图片数 >= img_count
   - 选择差值最小的模板（最优匹配）
4. 填充内容到匹配的模板
5. 清除未使用的占位符
```

## 三、占位符规则

### 3.1 文本占位符

| 占位符 | 用途 |
|--------|------|
| `{{h0_0}}` | 封面标题 |
| `{{h1_0}}` | 章节标题 |
| `{{h2_0}}` | 小节标题 |
| `{{h3_0}}`~`{{h3_3}}` | 文本内容（最多4个） |

### 3.2 多媒体占位符

| 占位符 | 用途 |
|--------|------|
| `{{img_0}}`~`{{img_4}}` | 图片（最多5张） |
| `{{audio_0}}` | 音频 |
| `{{video_0}}` | 视频 |

---

## 四、清除多余内容

### 4.1 清除规则

填充内容后，自动清除未使用的占位符：

```python
# 已使用的占位符
used = ['h2_0', 'h3_0', 'h3_1']  # 小节名 + 2个文本框

# 清除未使用的
clear_unused(used)  # 清除 h3_2, h3_3, img_0, img_1...
```

### 4.2 清除方式

将占位符shape移动到不可见位置：
```python
shape.left = Emu(0)
shape.top = Emu(0)
shape.width = Emu(0)
shape.height = Emu(0)
```

## 五、拼音生成规则

### 5.1 垂直对齐原则
- **核心要求**：每个拼音精确对齐到对应汉字的**正上方**
- 拼音和汉字**一一对应**

### 5.2 表格实现方案
使用 **2行 x N列** 的无边框、无底色表格：

```
原文：这是白色
实现：
 zhè  shì  bái  sè
 这    是    白    色
```

### 5.3 字符间距规则
- **拼音间距**：每个拼音后 + 1个字母宽度
- **汉字间距**：跟随对应拼音
- **列宽计算**：动态计算，不均分

### 5.4 字体设置

| 行 | 字体 | 说明 |
|----|------|------|
| 拼音行 | Arial | 英文字体 |
| 汉字行 | SimSun | 宋体 |
| 字号 | 相同 | 自动缩放 |

### 5.5 表格样式
- **边框**：完全隐藏
- **底色**：透明
- **行高**：拼音45%，汉字55%
- **换行**：禁止（`word_wrap = False`）

### 5.6 自动缩放
- 文本过长时自动缩小字号
- 最小字号：10号

## 六、模板匹配流程

```
1. 分析模板库
   ├─ 识别封面页、章节页
   └─ 统计正文页的文本框数和图片数

2. 解析MD文档
   ├─ 提取课程标题（封面）
   └─ 提取各小节内容（正文）

3. 逐页匹配
   ├─ 封面 → 封面模板
   └─ 正文 → 最佳匹配模板

4. 填充内容
   ├─ 替换文本占位符（拼音表格）
   └─ 保留图片等元素

5. 清理页面
   └─ 清除未使用的占位符

6. 输出PPT
```
