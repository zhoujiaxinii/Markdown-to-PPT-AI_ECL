# MD撰写规范

> 更新日期：2026-03-04

---

## 一、文档结构（3级标题）

| 级别 | 语法 | 用途 | 示例 |
|------|------|------|------|
| 一级 | `# 标题` | 课程标题 | `# 说白色` |
| 二级 | `## 标题` | 章节名 | `## 词汇` |
| 三级 | `### 标题` | 小节名 | `### 学一学` |

**重要**：每个小节（`###`）后的内容会生成**一页PPT**。

---

## 二、文本框

### 2.1 单文本框
```markdown
### 学一学
这是文本内容。
```

### 2.2 多文本框（最多4个）

用 `||` 分隔不同文本框的内容：

```markdown
### 学一学
A：这是什么颜色？||B：这是白色。
```

**效果**：生成一页PPT，包含两个文本框。

### 2.3 字数限制

| 字数范围 | 推荐文本框数 | 说明 |
|----------|-------------|------|
| 0-50字 | 1个 | 简短内容 |
| 51-100字 | 1-2个 | 中等内容 |
| 101-200字 | 2-3个 | 较长内容 |
| 201-400字 | 3-4个 | 内容较多 |
| 400字+ | 建议分页 | 内容过多 |

## 三、图片（0-5张/页）

```markdown
![图片](https://example.com/image.jpg)
![图片1](https://example.com/img1.jpg)
![图片2](https://example.com/img2.jpg)
```

---

## 四、音频（mp3）- 与文本同页

```markdown
### 听录音
请听录音判断对错。

![音频](https://example.com/audio.mp3)
```

**效果**：匹配带喇叭图标的音频模板。

---

## 五、视频（mp4）- 单独一页

```markdown
### 看视频
看视频回答问题。

![视频](https://example.com/video.mp4)
```

**效果**：生成单独一页，视频嵌入显示。

---

## 六、拼音标注规则

### 6.1 自动生成
所有汉字会自动添加拼音标注，拼音显示在汉字正上方。

### 6.2 对齐方式
- 使用表格实现精确对齐
- 每个拼音-汉字对在独立单元格
- 拼音间距：1个字母宽度
- 字号自动缩放（最小10号）

### 6.3 效果示意
```
 zhè  shì  bái  sè
 这    是    白    色
```

## 七、完整示例

```markdown
# 说白色

## 今天学什么

### 学一学
A：这是什么颜色？||B：这是白的。

![图片](https://example.com/white.jpg)

## 词汇

### 学一学
白

![图片](https://example.com/bai.jpg)

白衣服

![图片](https://example.com/clothes.jpg)

### 听录音
请听录音判断对错。

![音频](https://example.com/audio.mp3)

### 看视频
看视频回答问题。

![视频](https://example.com/video.mp4)

## 复习

### 知识点清单
白色、是字句
```

## 八、注意事项

| 项目 | 限制 |
|------|------|
| 标题层级 | 必须使用 `###` 定义小节 |
| 文本框/页 | 最多4个（用 `||` 分隔） |
| 图片/页 | 最多5张 |
| 拼音 | 自动生成，无需手动添加 |
