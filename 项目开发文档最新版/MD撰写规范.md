# MD撰写规范

> 更新日期：2026-03-10

---

## 一、文档结构（3级标题）

| 级别 | 语法 | 用途 | 示例 |
|------|------|------|------|
| 一级 | `# 标题` | 课程标题 | `# 说白色` |
| 二级 | `## 标题` | 章节名 | `## 词汇` |
| 三级 | `### 标题` | 小节名 | `### 学一学` |

**重要**：每个小节（`###`）后的内容会生成**一页PPT**。

---

## 二、文本框

### 2.1 单文本框
```markdown
### 学一学
这是文本内容。
```

### 2.2 多文本框（最多4个）

用 `||` 分隔不同文本框的内容：

```markdown
### 学一学
A：这是什么颜色？||B：这是白色。
```

**效果**：生成一页PPT，包含两个文本框。

### 2.3 字数限制

| 字数范围 | 推荐文本框数 | 说明 |
|----------|-------------|------|
| 0-50字 | 1个 | 简短内容 |
| 51-100字 | 1-2个 | 中等内容 |
| 101-200字 | 2-3个 | 较长内容 |
| 201-400字 | 3-4个 | 内容较多 |
| 400字+ | 建议分页 | 内容过多 |

## 三、图片（0-5张/页）

```markdown
![图片](https://example.com/image.jpg)
![图片1](https://example.com/img1.jpg)
![图片2](https://example.com/img2.jpg)
```

## 四、音频（MP3）

### 4.1 撰写语法

音频文件使用以下语法引入：

```markdown
### 听录音
请听录音判断对错。

https://example.com/audio.mp3
```

**或者与文本同页**：

```markdown
### 学一学
A：听录音回答问题。||B：我听懂了。

https://example.com/audio.mp3
```

**说明**：
- 音频文件链接必须是 **在线URL**（以 `.mp3`、`.wav`、`.m4a` 结尾）
- 音频可以**单独占一页**，也可以与文本框**同页**
- 与音频同页时，匹配含 `{{audio}}` 占位符的模板页
- 嵌入音频，失败时显示红色错误提示

### 4.2 支持的格式

| 格式 | 结尾示例 |
|------|----------|
| MP3 | `.mp3` |
| WAV | `.wav` |
| M4A | `.m4a` |

---

## 五、视频（MP4）

### 5.1 撰写语法

视频文件使用以下语法引入：

```markdown
### 看视频
看视频回答问题。

https://example.com/video.mp4
```

**或者与文本同页**：

```markdown
### 学一学
看视频回答问题。||A：是的。

https://example.com/video.mp4
```

**说明**：
- 视频文件链接必须是 **在线URL**（以 `.mp4`、`.avi`、`.webm` 结尾）
- 视频可以**单独占一页**，也可以与文本框**同页**
- 与视频同页时，匹配含 `{{video}}` 占位符的模板页
- 嵌入视频，提取第一帧作为封面，失败时显示红色错误提示
- 若找不到合适模板，输出时报错："有内容未找到合适的模板匹配"

### 5.2 支持的格式

| 格式 | 结尾示例 |
|------|----------|
| MP4 | `.mp4` |
| AVI | `.avi` |
| WebM | `.webm` |

## 六、字体规则

### 6.1 字体统一
- 数字、英文、汉字、拼音统一使用**黑体**（SimHei）

### 6.2 字体大小
根据文本框数量自动调整，同一页多框时大小一致：

| 文本框数 | 字体大小 |
|----------|----------|
| 1个 | 48pt |
| 2个 | 36pt |
| 3个 | 30pt |
| 4个 | 24pt |

---

## 七、拼音标注规则

### 6.1 自动生成
所有汉字会自动添加拼音标注，拼音显示在汉字正上方。

### 6.2 对齐方式
- 使用表格实现精确对齐
- 每个拼音-汉字对在独立单元格
- 拼音间距：1个字母宽度
- 字号自动缩放（最小10号）

### 6.3 效果示意
```
 zhè  shì  bái  sè
 这    是    白    色
```

## 九、注意事项

| 项目 | 限制 |
|------|------|
| 标题层级 | 必须使用 `###` 定义小节 |
| 文本框/页 | 最多4个（用 `||` 分隔） |
| 图片/页 | 最多5张 |
| 音频 | 单独一页或与文本同页，不能超过4个文本框 |
| 视频 | 单独一页或与文本同页，不能超过4个文本框 |
| 拼音 | 自动生成，无需手动添加 |
