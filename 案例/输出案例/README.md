# 输出案例

> 存放每次迭代生成的PPT案例，按版本号+时间命名

## 命名规则

```
{版本号}_{日期}_{时间}_{课程名}.pptx
{版本号}_{日期}_{时间}_{课程名}.md      ← 对应的源MD文件
```

示例：`V12_2026-03-05_0300_颜色世界.pptx`
