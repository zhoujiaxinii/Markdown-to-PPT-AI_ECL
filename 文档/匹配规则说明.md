# 匹配规则说明

> 详细说明MD内容如何匹配到PPT模板的规则。

---

## 一、页面类型识别

系统根据MD内容自动识别页面类型，优先级如下：

```
视频页面（最高） > 音频页面 > 普通正文页面 > 章节页面 > 目录页面 > 封面
```

---

## 二、页面类型与模板匹配

### 2.1 封面页
- **识别条件**：`# 一级标题`
- **模板占位符**：`{{h0_0}}`
- **示例**：`# 英语语法精讲`

### 2.2 目录页
- **识别条件**：`## 二级标题` 列表
- **模板占位符**：`{{h1_0}}`, `{{h1_1}}`, `{{h1_2}}`...
- **示例**：
  ```markdown
  ## 第一章
  ## 第二章
  ## 第三章
  ```

### 2.3 章节页
- **识别条件**：单独的 `## 二级标题`（后面没有 `###` 小节）
- **模板占位符**：`{{h1_0}}`
- **示例**：`## 语法讲解`

### 2.4 普通正文页
- **识别条件**：`### 小节名` + 文本内容
- **模板占位符**：
  - 小节名：`{{h2_0}}`
  - 文本框：`{{h3_0}}`, `{{h3_1}}`, `{{h3_2}}`, `{{h3_3}}`（最多4个）
  - 图片：`{{img_0}}` ~ `{{img_4}}`（最多5张）

### 2.5 音频页面
- **识别条件**：`### 小节名` + `![音频](url)` + 文本内容
- **模板特征**：包含喇叭图标占位符 `{{audio_0}}`
- **模板占位符**：
  - 小节名：`{{h2_0}}`
  - 文本框：`{{h3_0}}`, `{{h3_1}}`
  - 音频：`{{audio_0}}`

### 2.6 视频页面
- **识别条件**：`### 小节名` + `![视频](url)`
- **模板特征**：包含视频占位符 `{{video_0}}`
- **模板占位符**：
  - 小节名：`{{h2_0}}`
  - 视频：`{{video_0}}`

---

## 三、文本框匹配规则

### 3.1 文本框数量计算
- 使用 `||` 分隔符将文本分成多个文本框
- 最多支持 **4个** 文本框

**示例**：
```markdown
### 学一学
第一段文本||第二段文本||第三段文本
```
→ 生成3个文本框的页面

### 3.2 文本框占位符对应
| 文本框序号 | 占位符 |
|-----------|--------|
| 第1个 | `{{h3_0}}` |
| 第2个 | `{{h3_1}}` |
| 第3个 | `{{h3_2}}` |
| 第4个 | `{{h3_3}}` |

---

## 四、图片匹配规则

### 4.1 图片数量计算
- 每个 `![图片](url)` 算一张图片
- 最多支持 **5张** 图片

### 4.2 图片占位符对应
| 图片序号 | 占位符 |
|---------|--------|
| 第1张 | `{{img_0}}` |
| 第2张 | `{{img_1}}` |
| 第3张 | `{{img_2}}` |
| 第4张 | `{{img_3}}` |
| 第5张 | `{{img_4}}` |

---

## 五、音视频识别规则

### 5.1 音频识别
- **关键词**：`![音频](url)`
- **URL后缀**：`.mp3`
- **页面行为**：与文本同页，使用带喇叭图标的模板

### 5.2 视频识别
- **关键词**：`![视频](url)`
- **URL后缀**：`.mp4`
- **页面行为**：单独一页，视频嵌入显示

---

## 六、拼音生成规则

### 6.1 垂直对齐原则
- **核心要求**：每个拼音精确对齐到对应汉字的**正上方**
- 拼音和汉字**一一对应**，位置对齐

### 6.2 表格实现方案
使用 **2行 x N列** 的无边框、无底色表格实现精确对齐：

```
原文：这是白色
实现：
 zhè  shì  bái  sè
 这    是    白    色
 ↑    ↑    ↑    ↑
每个拼音-汉字对在独立单元格中
```

### 6.3 字符间距规则
- **拼音间距**：每个拼音后 + 1个字母宽度的间距
- **汉字间距**：跟随对应拼音，自动对齐
- **列宽计算**：根据拼音长度动态计算，不再是均分

### 6.4 字体设置
| 行 | 字体 | 说明 |
|----|------|------|
| 拼音行 | Arial | 标准英文字体 |
| 汉字行 | SimSun | 宋体（中文等宽） |
| 字号 | 相同 | 拼音和汉字使用相同字号 |

### 6.5 表格样式
- **边框**：完全隐藏（无边框）
- **底色**：透明（无填充）
- **行高**：拼音行45%，汉字行55%
- **对齐**：水平居中 + 垂直居中

### 6.6 非汉字处理
- 英文字母、数字、标点：上方单元格为空
- 占据与汉字相同的单元格宽度
- 保持整体对齐

---

## 七、模板匹配流程

```
1. 读取MD文档
   ↓
2. 解析标题层级
   ↓
3. 遍历每个小节(###)
   ├─ 是否有视频？→ 匹配视频模板
   ├─ 是否有音频？→ 匹配音频模板
   └─ 普通内容 → 匹配普通正文模板
   ↓
4. 替换占位符
   ↓
5. 生成PPT
```
